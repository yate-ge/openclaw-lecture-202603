## Project Overview

This is a lecture/presentation project (not a software codebase). It contains slides for a university course aimed at design students (undergraduate and graduate), teaching AI Agent concepts from an interaction design perspective.

**Topic**: From LLMs to Agents — understanding OpenClaw's architecture and its implications for interaction design.

**Primary file**: `openclaw-slides-202603.md` — the complete slide deck in Markdown format, structured with `---` separators between sections.

## Content Structure

The slides follow a progressive narrative:

1. **Part 1** — Technical evolution: LLM → Chatbot → Agent (ReAct) → OpenClaw (always-on agent)
2. **Part 2** — Interaction paradigm shifts: proactive interaction, memory, end-user development, multi-channel UX, and design challenges (transparency, trust, error recovery, privacy, social dynamics)
3. **Part 3** — Practical demo: OpenClaw + Tami robot integration (has TODO items to be filled in)
4. **References** — Curated links organized by category (OpenClaw core, Agent fundamentals, interaction design, robotics, Chinese-language resources)
